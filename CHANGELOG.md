# Changelog

## [0.1.0] - 2026-07-16

### Added

- Initial repository structure
- Core philosophy documents
- Space and geometry specifications
- Spatial hierarchy specifications
- Space physics specifications
- Motion specifications
- Typography specifications
- Layout specifications
- Component specifications (Button, Input, Card, Modal, Navigation)
- Design token specifications (Color, Spacing, Border)
- Accessibility guidelines
- User psychology documentation
- Example documentation
- Anti-pattern documentation
- Best practices documentation
- Migration guides (Flat Design, Material Design)
- Glossary and FAQ
- RFC for design principles
- Case studies framework
- Templates for pages and components
- GitHub infrastructure (issues, PRs, workflows, labels)
- Contributing guidelines
- Code of Conduct
- Security policy
- MIT License

### Notes

- This is the initial release establishing the complete specification framework
- All specifications are technology-agnostic by design
- Documents are written to be comprehensible by developers, designers, and AI
