# Welcome to WHITE DIAMOND VS Code Extension

## What's in the folder

* This folder contains all of the files necessary for your extension.
* `package.json` - The manifest file in which you declare your extension and command.
* `src/extension.ts` - The main source file for your extension.

## Get up and running straight away

* Press `F5` to open a new window with your extension loaded.
* Open a CSS, SCSS, or LESS file.
* The extension will automatically detect violations.

## Make changes

* You can relaunch the extension from the debug toolbar after making changes to the files listed above.
* You can also reload (`Ctrl+R` or `Cmd+R` on Mac) the VS Code window with your extension to load your changes.

## Explore the API

* Open the file `src/extension.ts`.
* Add a new command to the `activate` function.
* Run the command in the Debug Console.

## Run tests

* Open the Debug Viewlet (`Ctrl+Shift+D` or `Cmd+Shift+D` on Mac).
* From the Debug dropdown, select "Extension Tests".
* Press `F5` to run the tests in a new window with your extension loaded.
