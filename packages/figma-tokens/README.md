# @white-diamond/figma-tokens

Figma Tokens for the WHITE DIAMOND design specification.

## Installation

```bash
npm install @white-diamond/figma-tokens
```

## Usage

Import the tokens in your project:

```javascript
import { colors, spacing, borders, typography, shadows, animation } from '@white-diamond/figma-tokens';
```

## Features

- **Colors**: Neutral, primary, and semantic color tokens
- **Spacing**: Consistent spacing scale
- **Borders**: Border definitions following WHITE DIAMOND principles
- **Typography**: Font sizes, weights, and line heights
- **Shadows**: Elevation system
- **Animation**: Motion tokens for physics-based animations

## Themes

- Light theme
- Dark theme

## WHITE DIAMOND Specification

This package implements the WHITE DIAMOND design specification, where:

1. **Space is infinite** - The screen is a window into a vast architectural space
2. **Objects have mass and boundaries** - Every element is an architectural construction
3. **Three-layer hierarchy** - Physical Boundary → Inner Surface → Content
4. **Only rectangles** - Circles, ovals, organic shapes are forbidden
5. **Borders are everything** - Without a border, an object doesn't exist
